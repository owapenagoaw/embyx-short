# EmbyX Short (Android)

EmbyX Short 是基于 Kotlin + Jetpack Compose 的安卓短视频流式播放器项目，面向 Emby 媒体库使用场景。

## 仓库信息

- 仓库名：`embyx-short`
- 应用包名：`com.lalakiop.embyx`
- 最低系统版本：Android 7.0+（minSdk 24）

## 已构建安装包

当前 Release 安装包位置：

```text
app/release/app-release.apk
```

## 本地构建

```bash
./gradlew assembleRelease
```

构建产物默认输出到：

```text
app/release/app-release.apk
```
